# XBRL을 활용한 통합 재무제표 엑셀 파일 추출기
# 원하는 상장 기업의 재무제표를 자동으로 추출하고 엑셀 파일로 저장하는 코드입니다.

## 사용법
1. 기업명 입력 : 원하는 상장기업의 기업명을 입력합니다.
2. API 키 입력 : 본인의 API 키를 입력합니다. (보안을 위해 별표시로 마킹처리 하였음)
3. Flatten 및 Cleansing : XBRL 데이터 특성상 컬럼명이 깔끔하게 뽑히지 아니하여
                          단일 층의 컬럼으로 평탄화 작업을 수행하고
                          평탄화 작업으로 인해 지저분해진 컬럼명을 깔끔하게 정리합니다.
4. 엑셀 통합 저장 : 모든 재무제표를 하나의 엑셀 파일 내 sheet별로 저장합니다.
